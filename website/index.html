Quelques mots sur City-Explorer.fr

City-explorer.fr est un projet open source basé sur l'open data. Le projet est fortement inspiré du site web insideairbnb.com
City Explorer est ouvert à toute collaboration, pour les dévelopeurs via la page github du projet ou simplement pour les curieux qui souhaite apprendre. Merci de noter que les informations affichées peuvent dans certaines conditions ne pas être exactes. Cela dépend en partie de la date de récupération des données.

l'open Data c'est quoi ?

L'open data est une pratique consistant à rendre des données accessibles à tous, sans restriction, afin de favoriser la transparence, l'innovation et le développement de services et d'applications utiles à la société.

Comment collaborer ?

Vous êtes designer, photographe, développeur, rédacteur, etc. ... Le projet est loin d'être parfait, toutes les contributions sont les bienvenues. Pour les développeurs, cela se passe sur la page Github du projet ici ou en envoyant un email à trouver un email ici
